# 【未来の自分へ】実装予定の機能一覧

このファイルは、プロジェクトの将来的な拡張機能やアイデアを記録するためのものです。

## 1. AIによる名刺画像の自動文字認識（OCR）機能

### 現状
- 現在のバージョンでは、コストとデプロイの制約（Firebase無料プランの仕様）により、Cloud Functionsを利用したバックエンド処理を**意図的に削除**しています。
- そのため、名刺登録はユーザーによる完全な手動入力となっています。

### 次のステップ
プロジェクトが収益化し、FirebaseのBlazeプラン（有料プラン）へ移行できるようになった時が、この機能を復活させる時です。

**実装の概要:**
1.  FirebaseプロジェクトをBlazeプランにアップグレードする。
2.  削除した`functions`ディレクトリを復活させ、Cloud Functionsを再デプロイする。
3.  フロントエンド(`creation.js`)から、アップロードされた画像をCloud Functionに渡す処理を復活させる。
4.  Cloud Function内で、`@google-cloud/vision` APIを呼び出し、画像からテキストを抽出する。
5.  抽出したテキストをフロントエンドに返し、各フォームに自動入力する。

この「宿題」を忘れないでください。この機能が、このアプリの価値を大きく飛躍させます。
